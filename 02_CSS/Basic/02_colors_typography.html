Color Systems

Understanding different ways to define colors in CSS is fundamental to web design.

1. Hexadecimal Colors

Six-digit codes representing RGB values (e.g., #FF6B6B)

#FF6B6B
#4ECDC4
#45B7D1
#96CEB4
#FFEAA7

2. RGB Colors

Red, Green, Blue values from 0-255

rgb(255, 107, 107) - Red
rgb(78, 205, 196) - Teal

3. HSL Colors

Hue, Saturation, Lightness - more intuitive

hsl(0, 100%, 74%) - Red hue
hsl(174, 72%, 56%) - Teal hue

4. Named Colors

CSS provides 147 named colors

red

Typography Fundamentals

Typography is the art of arranging type to make written language legible, readable, and appealing.

1. Font Families

Different categories of fonts:

Serif fonts have decorative strokes
Sans-serif fonts are clean and modern
Monospace fonts have fixed-width characters
Cursive fonts mimic handwriting
Fantasy fonts are decorative

2. Font Sizes

Different units for text size:

12px - Small text
1em - Base size
1.2rem - Larger
1.5em - Even larger
2rem - Heading size

3. Font Weights

Control text thickness:

300 - Light
400 - Normal
600 - Semi Bold
800 - Extra Bold

4. Text Alignment

Control text positioning:

Left aligned
Center aligned
Right aligned
Justified text spreads words evenly across the line.

Best Practices

Color Accessibility

  • Ensure sufficient contrast (4.5:1 for normal text)
  • Don't rely solely on color for information
  • Test with color blindness simulators
  • Use tools like WebAIM contrast checker

Typography Tips

  • Use web-safe fonts or provide fallbacks
  • Maintain consistent vertical rhythm
  • Limit to 2-3 font families per design
  • Use relative units (em, rem) for scalability

Performance

  • Optimize web font loading
  • Use font-display for better loading
  • Limit font weights and styles
  • Consider system fonts for performance

Design Principles

  • Create a consistent color palette
  • Use hierarchy to guide the eye
  • Maintain adequate white space
  • Test on different devices
