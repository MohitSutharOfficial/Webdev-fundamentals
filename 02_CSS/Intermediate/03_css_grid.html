üìö What is CSS Grid?

CSS Grid is a powerful 2-dimensional layout system that allows you to create complex layouts with rows and columns. Unlike Flexbox (which is 1-dimensional), Grid can handle both dimensions simultaneously.

üéØ When to Use Grid:

  • Complex 2D layouts (rows AND columns)
  • Page layouts with header, sidebar, main, footer
  • Card-based designs
  • Image galleries
  • Dashboard layouts

üìê Grid Template Properties

Grid Template Columns & Rows

/* Different ways to define columns */ grid-template-columns: 200px 1fr 200px; /* Fixed-Flexible-Fixed */ grid-template-columns: repeat(3, 1fr); /* 3 equal columns */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* Responsive */ /* Grid rows */ grid-template-rows: auto 1fr auto; /* Header-Main-Footer */ grid-template-rows: 100px 200px 100px; /* Fixed heights */

Grid Template Areas

.layout { display: grid; grid-template-columns: 200px 1fr 200px; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .aside { grid-area: aside; } .footer { grid-area: footer; }

üìè Grid Units & Sizing

The 'fr' Unit (Fractional Unit)

/* 'fr' represents a fraction of available space */ grid-template-columns: 1fr 2fr 1fr; /* 1:2:1 ratio */ grid-template-columns: 200px 1fr; /* Fixed + Flexible */ grid-template-columns: auto 1fr auto; /* Content-based + Flexible */

minmax() Function

/* Minimum 200px, maximum 1fr */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Minimum content size, maximum 300px */ grid-template-columns: minmax(min-content, 300px) 1fr;

üéØ Grid Sizing Best Practices:

  • fr units: Use for flexible sizing
  • minmax(): Use for responsive minimum/maximum constraints
  • auto: Use for content-based sizing
  • repeat(): Use to avoid repetition

üîÑ Responsive Grid Layouts

Auto-fit and Auto-fill

/* Auto-fit: Stretches items to fill container */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); /* Auto-fill: Maintains item size, creates empty columns */ grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

Grid Line Placement

‚ö° Advanced Grid Techniques

1. Implicit Grid

.grid { display: grid; grid-template-columns: repeat(3, 1fr); /* Auto-generated rows */ grid-auto-rows: 100px; grid-auto-flow: row | column | dense; }

2. Grid Gaps

.grid { gap: 20px; /* Both row and column gap */ row-gap: 20px; /* Row gap only */ column-gap: 30px; /* Column gap only */ /* Old syntax (still supported) */ grid-gap: 20px; grid-row-gap: 20px; grid-column-gap: 30px; }

3. Dense Grid Packing

.grid { grid-auto-flow: dense; /* Fill gaps with smaller items */ }

üéØ Senior Developer Tips:

  • Use CSS Grid for 2D layouts - Perfect for complex page layouts
  • Combine with Flexbox - Use Grid for overall layout, Flexbox for components
  • Consider IE11 support - Use @supports for progressive enhancement
  • Use grid-template-areas - Makes code more readable and maintainable
  • Test responsive behavior - Always test auto-fit/auto-fill layouts

üîß Grid vs Flexbox

When to Use Grid vs Flexbox:

Use CSS Grid When:

  • 2D layouts (rows AND columns)
  • Complex page layouts
  • Overlapping content
  • Different sized grid items
  • Layout-first design

Use Flexbox When:

  • 1D layouts (row OR column)
  • Component-level layouts
  • Centering content
  • Equal-height columns
  • Content-first design

üö® Common Pitfalls & Solutions

‚ö†Ô∏è Common Mistakes:

  • Forgetting explicit grid placement: Items may not appear where expected
  • Using Grid for 1D layouts: Flexbox is usually better
  • Not testing responsive behavior: auto-fit/auto-fill can be tricky
  • Overusing fixed units: Use fr units for flexibility
  • Ignoring grid-area names: They make code much more maintainable

‚úÖ Best Practices:

  • Use meaningful grid-area names
  • Test layouts on different screen sizes
  • Use CSS Grid Inspector in dev tools
  • Keep layouts simple and intuitive
  • Use progressive enhancement for older browsers
