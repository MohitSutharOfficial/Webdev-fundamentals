CSS Flexbox Layout System

Flexbox (Flexible Box Layout) is a powerful CSS layout method that makes it easy to design flexible and responsive layout structures. It's perfect for one-dimensional layouts and component-level design patterns.

1. Basic Flexbox Concepts

Flex Container (Parent)

The flex container is the parent element with `display: flex`. It establishes the flex formatting context for its children.

.flex-container { display: flex; /* flex-direction: row; (default) */ /* flex-wrap: nowrap; (default) */ /* justify-content: flex-start; (default) */ /* align-items: stretch; (default) */ }

7. Common Flexbox Patterns

🎯 Senior Engineer Tips:

  • Perfect Centering: Use `display: flex; justify-content: center; align-items: center;`
  • Equal Height Columns: Flexbox automatically creates equal heights
  • Sticky Footer: Use `flex: 1` on main content to push footer down
  • Responsive Navigation: Switch from row to column on mobile
  • Media Object: Perfect for comments, user profiles, notifications

8. When to Use Flexbox vs CSS Grid

Use Flexbox When:

  • You need to align items in one dimension (row OR column)
  • Content size determines layout (content-first)
  • You need to distribute space between items
  • Building components like navigation, cards, media objects

Use CSS Grid When:

  • You need to control both rows and columns (2D layout)
  • Layout should be defined first (layout-first)
  • You're building page-level layouts
  • You need precise control over item placement

9. Common Flexbox Gotchas

⚠️ Common Issues:

  • Min-width on flex items: Items won't shrink below their content size by default
  • Margin collapse: Margins don't collapse in flexbox
  • Percentage heights: May not work as expected in flex containers
  • Z-index stacking: Flex items create new stacking contexts
/* Fix for items not shrinking */ .flex-item { min-width: 0; /* Allow item to shrink below content size */ overflow: hidden; /* Prevent content overflow */ } /* Fix for text wrapping in flex items */ .flex-item { word-wrap: break-word; overflow-wrap: break-word; }
