<!-- Advanced media queries --> <style> /* High-resolution displays */ @media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 2dppx) { .high-res-image { background-image: url('image@2x.jpg'); } } /* Orientation-based styles */ @media (orientation: landscape) { .landscape-only { display: block; } } @media (orientation: portrait) { .portrait-only { display: block; } } /* Hover capability detection */ @media (hover: hover) { .hover-effects:hover { transform: scale(1.05); } } /* Reduced motion preference */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } } /* Dark mode support */ @media (prefers-color-scheme: dark) { body { background-color: #1a1a1a; color: #ffffff; } } </style>

üöÄ Senior Engineer Tips:

  • Lazy Loading: Use `loading="lazy"` for images below the fold
  • WebP Support: Use Picture element to serve modern formats with fallbacks
  • Intersection Observer: Implement custom lazy loading for better control
  • Performance Budget: Monitor image sizes and optimize delivery
  • Accessibility: Always provide meaningful alt text and captions

‚ö†Ô∏è Common Pitfalls:

  • Not providing fallbacks for older browsers
  • Forgetting to optimize images for different screen sizes
  • Using fixed aspect ratios without considering content
  • Not testing on real devices with slow connections
  • Ignoring accessibility requirements for multimedia
