Key Concepts

1. Content Modification Methods

Different ways to modify element content:

// Text content (safe from XSS) element.textContent = 'Safe text content'; // HTML content (can execute scripts) element.innerHTML = '<strong>Bold text</strong>'; // Full element replacement element.outerHTML = '<div class="new">New element</div>'; // Visible text only element.innerText = 'Visible text only'; // Form element values inputElement.value = 'New input value';

2. Attribute Manipulation

Working with element attributes:

// Set attribute element.setAttribute('data-id', '123'); // Get attribute const value = element.getAttribute('data-id'); // Remove attribute element.removeAttribute('data-id'); // Check if attribute exists if (element.hasAttribute('data-id')) { console.log('Attribute exists'); } // Direct property access element.id = 'new-id'; element.className = 'new-class'; element.title = 'Tooltip text';

3. Style Manipulation

Modifying element styles:

// Individual style properties element.style.color = 'red'; element.style.backgroundColor = '#f0f0f0'; element.style.fontSize = '18px'; // Multiple styles at once element.style.cssText = 'color: red; background: yellow;'; // Getting computed styles const computedStyle = getComputedStyle(element); const color = computedStyle.color; // Removing styles element.style.removeProperty('color'); element.style.color = ''; // Alternative method

4. Class Management

Working with CSS classes:

// Add class element.classList.add('new-class'); // Remove class element.classList.remove('old-class'); // Toggle class element.classList.toggle('active'); // Check if class exists if (element.classList.contains('active')) { console.log('Element has active class'); } // Replace class element.classList.replace('old-class', 'new-class'); // Multiple classes element.classList.add('class1', 'class2', 'class3');

5. Security Considerations

Always be careful with user input:

// UNSAFE - Can execute malicious scripts element.innerHTML = userInput; // SAFE - Treats as plain text element.textContent = userInput; // SAFE - Sanitize HTML input function sanitizeHTML(html) { const div = document.createElement('div'); div.textContent = html; return div.innerHTML; } // Use with caution and validation element.innerHTML = sanitizeHTML(userInput);
